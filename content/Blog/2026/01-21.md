### 验证ssl与加载

#### 进入nginx容器验证与加载

- 查看nginx容器id

```
sudo docker ps
```

- 进入容易验证加载
```
sudo docker exec -it 560ab38bf8b8 /bin/bash
nginx -t
nginx -s reload
```

#### 主机上直接验证与加载

```
docker exec 560ab38bf8b8 nginx -t
docker exec 560ab38bf8b8 nginx -s reload
```
